Creates an EDT instance.

Parameters
guidReturned value: GUID of the newly created EDT type
templateGuidGUID of the template to use for this EDT
paramcNumber of non-DB 64 bit values. Set to 'EDT_PARAM_DEF' if follows the 'paramc' template specification. Set to the actual number of arguments to be passed if 'EDT_PARAM_UNK' has been given as the template's 'paramc' value.
paramvValues for those parameters (copied in)
propertiesUsed to indicate if this is a finish EDT (EDT_PROP_FINISH). Other uses reserved.
affinityAffinity container for this EDT. Can be NULL_GUID
depcNumber of dependences for this EDT. Set to 'EDT_PARAM_DEF' if it follows the 'depc' template specification. Set to the actual number of arguments to be passed if 'EDT_PARAM_UNK' has been given as the template's 'depc' value.
depvValues for the GUIDs of the dependences (if known) Use ocrAddDependence to add unknown ones or ones with a mode other than the default DB_MODE_ITW
outputEventReturned value: If not NULL, will return the GUID of the event that will be satisfied when this EDT completes. In the case of a finish EDT, this event will be satisfied only if all transitively created EDTs inside this EDT have also completed. If NULL, no output event will be created or satisfied
Returns
