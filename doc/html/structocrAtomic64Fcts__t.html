u64(* ocrAtomic64Fcts_t::cmpswap)(struct _ocrAtomic64_t *self, u64 cmpValue, u64 newValue)

Compare and swap.

The semantics are as follows (all operations performed atomically):

  • if location is cmpValue, atomically replace with newValue and return cmpValue
  • if location is not cmpValue, return value at location
