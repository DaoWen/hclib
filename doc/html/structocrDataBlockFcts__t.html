void*(* ocrDataBlockFcts_t::acquire)(struct _ocrDataBlock_t *self, ocrGuid_t edt, bool isInternal)

Acquires the data-block for an EDT.

This call registers a user (the EDT) for the data-block

Parameters
selfPointer for this data-block
edtEDT seeking registration
isInternalTrue if this is an acquire implicitly done by the runtime at EDT launch
Returns
Address of the data-block
Note
Multiple acquires for the same EDT have no effect

u8(* ocrDataBlockFcts_t::free)(struct _ocrDataBlock_t *self, ocrGuid_t edt)

Requests that the block be freed when possible.

This call will return true if the free request was successful. This does not mean that the block was actually freed (other EDTs may be using it), just that the block will be freed at some point

Parameters
selfPointer to this data-block
edtEDT seeking to free the data-block
Returns
0 on success and an error code on failure (see ocr-db.h)
