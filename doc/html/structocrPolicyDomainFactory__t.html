Create a policy domain.

Allocates the required space for the policy domain based on the counts passed as arguments. The 'schedulers', 'workers', 'computes', 'workpiles', 'allocators' and 'memories' data-structures must then be properly filled

Parameters
factoryThis policy domain factory
configurationAn optional configuration
schedulerCountThe number of schedulers
workerCountThe number of workers
computeCountThe number of compute targets
workpileCountThe number of workpiles
allocatorCountThe number of allocators
memoryCountThe number of memory targets
taskFactoryThe factory to use to generate EDTs
taskTemplateFactoryThe factory to use to generate EDTs templates
dbFactoryThe factory to use to generate DBs
eventFactoryThe factory to use to generate events
contextFactoryThe factory to use to generate context
guidProviderThe provider of GUIDs for this policy domain
costFunctionThe cost function used by this policy domain
